Renjin is a new interpeter for the R language that runs on the JVM. It's principle goal is to facilitate the use of R-language programs and libraries in a wider variety of environments, such as Google AppEngine, on which this example is running.

Renjin is meant to be 100% compatible with R 2.10+, but not all primitive functions are implemented. (help wanted!). If you see a result that is inconsistent with the original interpreter or just plain wrong, please report it.

This REPL is based on Google's LotREPLs.
